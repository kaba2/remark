DocChildren macro
=================

[[Parent]]: macros.txt

The DocChildren macro generates a list of links to the documentation
children of the currently processed document. This list is preceded 
by a level-1 heading. The heading is not generated if there are no
documentation children.

Properties
----------

### Output type

Remark

### Output expansion default

Disabled.

### Parameters

Ignored.

### Variables

DocChildren.class_name
: 	The name of the html-class to give to the `<div>` tag. The html-class 
	allows to style the generated list using CSS.
	Default 'DocChildren'.

DocChildren.title
:	The title of the heading. Default 'Learn more'.

DocChildren.no_links_for
: 	A list of links to not include in the generation. Default empty.

Example
-------

This

	[[DocChildren]]
	
produces

[[DocChildren]]

i.e. nothing because this document has no children.
For a more interesting example see the parent document
of this document.
