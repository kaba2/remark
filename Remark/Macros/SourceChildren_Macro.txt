SourceChildren macro
====================

[[Parent]]: macros.txt

Description
-----------

### Action

Generates a list of links to the source code
children of the currently processed document.
This list is preceded by a level-1 heading.

### Output type

Remark

### Output expansion default

Disabled.

### Parameters

Title. Default 'Files'.

### Variables

None

Example
-------

This

	[[SourceChildren]]
	
produces the following section (we don't want to duplicate it here).
