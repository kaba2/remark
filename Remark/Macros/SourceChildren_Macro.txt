SourceChildren macro
====================

[[Parent]]: macros.txt

Description
-----------

### Action

Generates a list of links to the source code
children of the currently processed document.
This list is preceded by a level-1 heading.

### Output type

Remark

### Parameters

Ignored.

### Variables

'SourceChildren:title': The title of the heading. Default 'Files'.

'SourceChildren:no_links_for': A list of links to not include in the generation. Default empty.

Example
-------

This

	[[SourceChildren]]
