Body macro
==========

[[Parent]]: macros.txt

Description
-----------

This macro is used internally by Remark, and should not be
used by normal users.

### Action

Reads the currently considered document from a file. Returns its contents.

### Output type

Remark

### Parameters

Ignored.

### Variables

None.

Example
-------

This

	[[Body]]

would lead to an an infinite macro expansion loop.
This macro is used internally by Remark, and should not be
used by normal users.
