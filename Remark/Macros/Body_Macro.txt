Body macro
==========

[[Parent]]: macros.txt

Description
-----------

### Action

Reads the currently considered document from a file. Returns its contents.

### Output type

Remark

### Output expansion default

Enabled.

### Parameters

Ignored.

### Variables

None.

Example
-------

This

	[[Body]]

would lead to an an infinite macro expansion loop.
This form of the macro invocation is seen only on
document templates. For demonstration purposes,
we can use:

	[[-+Verbatim]]: [[-Body]]

which produces this:

[[-+Verbatim]]: [[-Body]]
