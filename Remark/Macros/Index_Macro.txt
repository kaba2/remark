Index macro
===========

[[Parent]]: macros.txt

Description
-----------

### Action

Generates a directory listing from the files
and directories in the same directory as the
currently processed document file. Each directory
and file is given as a link.

### Output type

Remark

### Output expansion default

Disabled.

### Parameters

Ignored.

### Variables

None.
