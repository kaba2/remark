Comment macro
=============

[[Parent]]: macros.txt

Description
-----------

The intent of this macro is to allow commenting of the Remark source.

### Action

Returns nothing.

### Output type

Remark

### Parameters

Ignored.

### Variables

None.

Example
-------

This

	[[Comment]]:
		Place some explanation here.
		Do use more than one line if you want.
		
		Still a comment.
		
produces:

[[Comment]]:
	Place some explanation here.
	Do use more than one line if you want.
	
	Still a comment.

i.e. nothing.
