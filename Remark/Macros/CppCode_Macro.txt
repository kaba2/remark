CppCode macro
=============

[[Parent]]: macros.txt

Description
-----------

### Action

Color-encodes C and C++ source code. Outputs corresponding
html code. 

### Output type

Html

### Output expansion default

Disabled.

### Parameters

The parameter is interpreted as the input C++ source code.
If the parameter is not given, the content is loaded from 
the currently processed document file. 

### Variables

None.

### Details

The color-encoding is carried out by the Pygments Python library.
Note that Markdown syntax allows for inlined html.

Example
-------

This

	[[CppCode]]:
		int square(int x)
		{
			return x * x;
		}

produces this:
		
[[CppCode]]:
	int square(int x)
	{
		return x * x;
	}
