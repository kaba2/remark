CppCode macro
=============

[[Parent]]: macros.txt

Description
-----------

### Action

Highlights C++ source code. Outputs corresponding html code. 

### Output type

Html

### Output expansion default

Disabled.

### Parameters

The C++ source code to highlight.

### Variables

None.

### Details

The color-encoding is carried out by the Pygments library.

Example
-------

This

	[[CppCode]]:
		int square(int x)
		{
			return x * x;
		}

produces this:
		
[[CppCode]]:
	int square(int x)
	{
		return x * x;
	}
