Link macro
==========

[[Parent]]: macros.txt

Description
-----------

### Action

Generates links to documents in the
documentation tree. The labels for the links are set
to the descriptions of the target documents. 

### Output type

Remark

### Output expansion default

Disabled.

### Parameters

A list of target document paths for which the links are to be generated.

### Variables

None.

Example
-------

This
	
	[[Link]]: 
		macros.txt
		remark_syntax.txt
		../../remark_syntax.txt
		directory.index
	
produces this:

[[Link]]: 
	macros.txt
	remark_syntax.txt
	../../remark_syntax.txt
	directory.index
