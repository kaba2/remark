SkipExpansion macro
===================

[[Parent]]: macros.txt

Description
-----------

This macro can be used to prohibit macro
expansion from taking place over a region
of text. 

### Action

Copies the input as it is.

### Output type

Remark

### Output expansion default

Disabled.

### Parameters

The text to copy.

### Variables

None.

Examples
--------

This

	[[SkipExpansion]]:
		[[Link]]: macros.txt
		
		Some more text.
		
produces this:

[[SkipExpansion]]:
	[[Link]]: macros.txt

	Some more text.
