DirectoryLink macro
===================

[[Parent]]: macros.txt

Description
-----------

### Action

Generates links to the containing directories of documents 
in the documentation tree.  

### Output type

Remark

### Output expansion default

Disabled.

### Parameters

A list of target document paths for which the links are to be generated.

### Variables

None.

Example
-------

This
	
	[[DirectoryLink]]: 
		macros.txt
		remark_syntax.txt
		../../remark_syntax.txt
		directory.index
	
produces this:

[[DirectoryLink]]: 
	macros.txt
	remark_syntax.txt
	../../remark_syntax.txt
	directory.index
