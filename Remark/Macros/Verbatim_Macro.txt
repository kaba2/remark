Verbatim macro
==============

[[Parent]]: macros.txt

Description
-----------

### Action

Inserts a tab in front of each line of the
input parameter, thus reinterpreting text
as verbatim content. 

### Output type

Remark

### Output expansion default

Disabled.

### Parameters

The text to reinterpret.

### Variables

None.

Examples
--------

This

	[[+Verbatim]]: [[description]]
		
produces this:

[[+Verbatim]]: [[description]]
