Document types
==============

[[Parent]]: user_documentation.txt

The document types are registered at the start of Remark.
Here we list the currently supported document types. 

### Remark documentation file

Input extension
: 	.txt

Output extension
: 	.htm

AsciiMath interpretation
:	Enabled

Document template
: 

		[[Body]]
	    [[DocChildren]]
	    [[get mid_text]]
	    [[SourceChildren]]
	    [[get end_text]]

The purpose of the get macros is to enable the documentation
writer some freedom over positioning. For example, the
following can be used to place a link list at the end:

	[[set end_text]]:
		Links
		-----
		
		[[Link]]: some_file.txt

### C++ source-code file

We have packed several document types together here since
they act similarly. 

Input extensions 
:	.cpp, .cc, .hpp, hh, .c, .h

Output extensions (respectively)
:	.cpp.htm, .cc.htm, .hpp.htm, .hh.htm, .c.htm, .h.htm

AsciiMath interpretation
: Disabled

Document template
:	
		
		[[get file_name]]
		===
	
		[[Parent]]
	
		[[Link]]: directory.index
	
		[[-+CppCode]]: [[-Body]]

### Generic source-code file

Input extensions
:	.py, .m

Output extensions (respectively)
:	.py.htm, .m.htm

AsciiMath interpretation
:	Disabled

Document template
:	
		
		[[get file_name]]
		===
	
		[[Parent]]
	
		[[Link]]: directory.index
	
		[[-+GenericCode]]: [[-Body]]

### Directory view

Input extension
: .index

Output extension
: .htm

AsciiMath interpretation
:	Disabled

Document template
: 

		[[get description]]
		===
		
		[[Index]]
	
### Orphan list

Input Extension
: .orphan

Output extension
: .htm

AsciiMath interpretation
:	Disabled

Document template
: 

		Orphans
		=======
		
		[[DocChildren]]
		[[SourceChildren]]
