Third-party libraries
---------------------

### [Python Markdown][] (Python)

Python Markdown is licensed under the [BSD license][BSD]. 

This library is used to convert (extended) Markdown to html code.

[Python Markdown]: https://pythonhosted.org/Markdown/

### [Pygments][] (Python)

Pygments is licensed under the [BSD license][BSD]. 

This library is used to generate the highlighting for source code.

[Pygments]: http://pygments.org/

### [Highslide JS][] (JavaScript)

Highslide JS is _not_ free for commercial use. 

This library is used by [[Link: Gallery_Macro.txt]] to view images in Remark.

 * The Highslide JS JavaScript-library is referred to in an html-page 
 if and only if one invokes the Gallery-macro to generate 
 that page.
 
 * The Highslide JS JavaScript-library is copied to the output
 directory if and only if some generated html-page refers to it.

[Highslide JS]: http://www.highslide.com/

### [Pillow][] (Python)

Pillow is _not_ free for commercial use. 

This library is used by the [[Link: Gallery_Macro.txt]] to generate thumbnails automatically.
 
[Pillow]: https://python-pillow.github.io/
