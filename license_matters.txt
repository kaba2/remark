Third-party libraries
---------------------

### Python Markdown (Python)

Python Markdown is licensed under the BSD license. This library is used to generate the html code from Markdown.

### Pygments (Python)

Pygments is licensed under the BSD license. This library is used to generate the highlighting for source code.

### Highslide (JavaScript)

Highslide is not free for commercial use. This library is used by [[Link: Gallery_Macro.txt]] to view images in Remark.

 * The Highslide javascript-library is referred to in an html-page 
 if and only if one invokes the Gallery-macro to generate 
 that page.
 
 * The Highslide javascript-library is copied to the output
 directory if and only if some generated html-page refers to it.

### Pillow (Python)

Pillow is not free for commercial use. This library is used by the [[Link: Gallery_Macro.txt]] to generate thumbnails automatically.
